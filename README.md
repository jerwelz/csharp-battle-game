Ein kleines rundenbasiertes Kampfspiel, das ich in C# entwickelt habe, um objektorientierte Programmierung zu üben.

## Funktionen

- Spieler gegen KI
- Zufälliger Schaden
- Heilfunktion
- Rundenbasiertes Kampfsystem

## Verwendete Technologien

- C#
- .NET
- Konsolenanwendung

## Projekt starten

```bash
dotnet run
```
